# Goodreads Amazon Puppeteer

## Setup 

### Install Node Version 14.8

To install node using
[Node Version Manager](https://github.com/nvm-sh/nvm)
Run: 
```
nvm use
```

### Install dependencies

```
npm install
```
### TypeScript
If you need to install typescript...
```
npm install -g typescript
```
### Compile application

```
tsc
```

### Run
```
node index.js
```
